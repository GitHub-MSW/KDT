# Cast

***

```java
int intValue = 65;
char charValue = (char) intValue;

intValue = charValue;
```
- Casting 형변환
- 강제 형변환 : 큰내용을 작은 그릇에 담을때
- 자동 형변환 : 작은내용을 큰그릇에 담을때 자동(묵시적 형변환) 
- 기본형 타입
- byte < short = char < int < long < float < double
- `char charValue = (char) intValue;`강제형변환
- `intValue = charValue;`자동형변환
