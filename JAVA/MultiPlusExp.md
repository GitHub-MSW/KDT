# MultiPlusExp

***

```java
int var1 = 1;
int var2 = 2;
int var3 = 3;

int result1 = var1 + var2 * var3;
System.out.println("result1:" + result1);
int result2 = (var1 + var2) * var3;
System.out.println("result2:" + result2);

▶️
result1:7
result2:9

```
- `int result1 = var1 + var2 * var3;`연산자우선순위
- `int result2 = (var1 + var2) * var3;`연산자우선순위
